## 개요
림버스 컴퍼니의 거울던전을 랜덤으로 진행하게 하는 프로그램입니다.

---

## 사용법

1. 림버스 컴퍼니의 거울던전을 실행한 뒤 편성창에서 exe 파일을 실행한다.  
2. 실행하면 수감자 등급과 에고의 등급이 지정되며, 해당 등급에 맞게 자유롭게 편성한다.  
3. 가호 또한 출력된 내용에 따라 설정한 뒤 `y`를 입력하여 진행한다.  
4. 키워드가 지정되며 해당 키워드를 선택한다. 에고 기프트는 선택에 따라 획득한다.  
5. 관측도 키워드와 등급이 지정되며 자유롭게 획득한다.  
6. 팩 선택 시 새로고침이 표시되면 새로고침을 진행한다.  
   - 예: `[2번팩]`이 나오면 왼쪽부터 두 번째 팩을 선택한다.  
7. 입장하면 가호에 따라 획득할 기프트가 지정되며, 안내된 대로 획득한다.  
8. 맵 이동은 위에서부터 순서를 계산합니다.
    -예: '[2번길]'의경우 위에서부터 2번째팩을들어가면됩니다
